缺少linux/limits.h头文件，RHEL/CentOS系列可以安装glic-headers，alpine需要安装包——linux-headers，命令：

```
apk add --no-cache linux-headers
```

然后再执行pip install uwsgi即可安装。

另一个方法是，alpine直接用apk安装uwsgi，命令：

```
apk add --no-cache uwsgi
```
